# TarkovTradingCards 🃏

This is a Chinese localization fork of [Chazut/TarkovTradingCards](https://github.com/Chazut/TarkovTradingCards), with AI-assisted translation of:
- 21 quest definition files under `csharp/TTC.Mod/Services/Quests/` — all 359 quest names, descriptions, notes, and success messages
- Card names and descriptions in `bundles/`

For the original English version, visit the upstream repository.
